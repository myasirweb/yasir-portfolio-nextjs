# 🌐 Muhammad Yasir — Personal Portfolio

A fast, SEO-first personal developer portfolio showcasing projects, skills, and experience. Built with Next.js and TypeScript.

🔗 **Live Site:** [yasir-portfolio-nextjs.vercel.app](https://yasir-portfolio-nextjs.vercel.app)
📂 **GitHub:** [github.com/myasirweb/yasir-portfolio-nextjs](https://github.com/myasirweb/yasir-portfolio-nextjs)

---

## ✨ Features

- Lightning fast — built with Next.js App Router
- SEO optimized — meta tags, Open Graph, sitemap
- Fully responsive — mobile, tablet & desktop
- Smooth animations and transitions
- Projects showcase with live links
- Skills & tech stack section
- Contact form
- Lighthouse score 90+

## 🚀 Getting Started

### Prerequisites
- Node.js >= 18
- npm >= 9

### Installation

```bash
git clone https://github.com/myasirweb/yasir-portfolio-nextjs.git
cd yasir-portfolio-nextjs
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
